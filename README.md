# HazardGuard Nepal – Bilingual Disaster Awareness Platform

HazardGuard Nepal is a comprehensive disaster awareness and preparedness website designed for Nepal. It provides real‑time alerts for weather, air quality, floods, and earthquakes, along with an interactive risk map, emergency contacts, and a smart AI chatbot. The interface is fully bilingual (English and Nepali) with a convenient toggle switch.

## Features

- **Bilingual Interface** – Toggle between English and Nepali.
- **Real‑time Data** – Live weather (OpenWeatherMap), air quality (IQAir), earthquake alerts (USGS), and flood warnings (mock data).
- **Interactive Risk Map** – Visualize earthquake, flood, and landslide risk zones.
- **AI Chatbot** – Ask about disasters, weather, or districts in either language.
- **Emergency Contacts** – One‑click call buttons for police, fire, ambulance, and disaster helpline.

## Tech Stack

- HTML5, CSS3, JavaScript (ES6)
- Leaflet.js for maps
- OpenWeatherMap, IQAir, USGS APIs
- OpenAI API (ChatGPT) with CORS proxy
- Fully responsive design

The website is deployed by netlify on https://hazardguard.netlify.app/
